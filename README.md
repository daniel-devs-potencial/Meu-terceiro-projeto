# Meu Terceiro Projeto

Este projeto é uma landing page responsiva, construída com **HTML** e **CSS**, focada na apresentação visual profissional de um serviço fictício de compras via mobile: **Easy Shopping Via Mobile**.

## 🚀 Sobre

A página simula uma solução moderna de compras pelo celular, destacando:
- Título chamativo e slogan,
- Descrição breve do serviço,
- Botão de chamada ("Get Started"),
- Navegação no topo para Home, Work, About e Contact,
- Imagem ilustrativa de destaque,
- Layout adaptável para diferentes telas e dispositivos.

## 📂 Estrutura do Projeto

- `index.html`: Estrutura principal da página, com seções de navegação, conteúdo à esquerda e imagem à direita.
- `css/style.css`: Estilos de layout, responsividade e tipografia.
- `img/`: Contém a imagem visual utilizada na interface.

## 💡 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- Google Fonts

## 🎯 Como executar

1. Clone este repositório:
    ```bash
    git clone https://github.com/daniel-devs-potencial/Meu-terceiro-projeto.git
    cd Meu-terceiro-projeto
    ```
2. Abra o arquivo `index.html` em seu navegador.

Pronto! Você verá a página responsiva simulando a landing page de um serviço de compras mobile.
